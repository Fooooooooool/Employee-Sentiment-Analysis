## Key Insights
- Positive sentiment dominates internal communication.
- Negative sentiment is limited and generally short-lived.
- Message frequency is a strong indicator of employee engagement.
- No employees exhibit sustained negative sentiment patterns.

## Recommendations
- Focus on monitoring persistent negative emotions, rather than isolated incidents.
- Organizations should encourage constructive communication and interaction among employees.
- Managers can use emotional trend changes as early warning indicators for employee support or intervention.


## Tools and Libraries
This project was implemented in Python with the following main libraries:

- **Pandas** for data manipulation  
- **NumPy** for numerical operations  
- **NLTK / VADER** for initial sentiment analysis  
- **Transformers** for RoBERTa-based sentiment refinement  
- **Scikit-learn** for predictive modeling  
- **Matplotlib / Seaborn** for visualization
